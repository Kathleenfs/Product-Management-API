# Product Management API

## Overview

Product Management API is a RESTful backend application developed with Java and Spring Boot. The project demonstrates how to design and build a production-ready API following software engineering best practices, including layered architecture, database versioning, authentication, validation, testing, and containerization.

This repository is part of a backend portfolio composed of multiple independent services that simulate real-world enterprise applications.

---

## Objectives

The main objectives of this project are:

* Demonstrate REST API development with Spring Boot.
* Apply Layered Architecture principles.
* Manage relational data using PostgreSQL.
* Version the database using Flyway.
* Implement authentication and authorization with JWT.
* Validate requests and handle exceptions.
* Document the API with Swagger/OpenAPI.
* Containerize the application using Docker.
* Apply automated tests.

---

## Main Features
